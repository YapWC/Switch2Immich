# Switch2Immich
It is a cross platform desktop client software that enable user to bulk uploads rather than drag & drop manually to upload all their digital memories like images & videos from their computer HDD, SSD, pendrive, etc to your personal immich account or instance.

# How to Guide
1. Login to Your Immich account and navigate to Account Settings and under API Key, generate New API Key and Copy.
2. Connect to Your Immich instance via API Key
